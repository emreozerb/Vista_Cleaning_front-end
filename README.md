# Vista Cleaning - Professionele Schoonmaakdiensten

Een moderne, premium website voor Vista Cleaning, een professioneel schoonmaakbedrijf gevestigd in Heusden-Zolder.

## Kenmerken

### Design
- **Merkkleuren**: Deep navy (#3B4A7C) en light blue (#6FA9C8) afgeleid van het Vista Cleaning logo
- **Premium uitstraling**: Moderne, schone esthetiek met subtiele animaties
- **Responsive**: Volledig geoptimaliseerd voor mobiel, tablet en desktop
- **Smooth animations**: Framer Motion voor vloeiende interacties

### Functionaliteiten
- **Hero sectie**: Grote hero met gradient achtergrond en call-to-action
- **Over ons**: Vier USP's met iconen en hover-effecten
- **Diensten**: Tabs voor Particulier en Zakelijk met service cards
- **Reviews**: Testimonials met 5-sterren beoordelingen
- **Contact formulier**: Volledig werkend formulier met validatie
- **Floating CTA**: Zwevende call-to-action knop die verschijnt bij scrollen
- **Scroll to top**: Knop om terug naar boven te scrollen
- **Smooth scroll**: Vloeiende navigatie tussen secties

### Technologie
- **Framework**: Next.js 13 met App Router
- **Styling**: Tailwind CSS
- **Componenten**: shadcn/ui
- **Animaties**: Framer Motion
- **Iconen**: Lucide React

```bash

# Start development server
npm run dev

# Build voor productie
npm run build

# Start productie server
npm start
```

De website draait op [http://localhost:3000](http://localhost:3000)
